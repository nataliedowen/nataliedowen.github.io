Contrast

Contrast Rebelion

contrastrebelion.com

Ironic that a website called contrast rebelion has great contrast? Not really, as this website is all about good contrast that makes text readbale.
The dark text contrast perfectly against the light background color, making the text pop, and making it so that you don't have to strain you eyes in order to figure out what the text is saying.

Pinterest

pinterest.com

When i think of alignment, i generally think of text, but here we can see a great example of alignment that has nothing to do with text. Everything on this page is aligned with something else on the page, and it is obvious that this page was set up using a grid system.
Everything looks clean, organized, and feels like it has a place - that is thanks to the alignment.
